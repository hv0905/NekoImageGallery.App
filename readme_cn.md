## 起步

### 准备

#### 所需工具

- git
- Node.js
- yarn
- Visual Studio Code（强烈推荐）或其他代码编辑器

#### 安装依赖项

1. 克隆此仓库
2. 使用yarn安装依赖项

   ```shell
   yarn install
   ```

### 运行开发服务器

使用以下命令运行vite开发服务器

```shell
yarn run dev
```

### 生产部署

1. 更改环境文件以匹配您的生产环境

   ```shell
   cp env/.env.example env/.env.production.local
   code env/.env.production.local
   ```

2. 以生产模式构建项目

   ```shell
   yarn run build
   ```

3. 将`dist`文件夹部署到您自己的Web服务器

### Lint

使用以下命令对项目进行lint

```shell
yarn run lint
```
